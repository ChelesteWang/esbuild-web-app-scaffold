一个基于 Esbuild 的现代 Web 开发脚手架。你甚至无需任何配置就能启动一个支持各种现代前端工程能力的项目。

## 快速开始

```js
// npm
npm init @ewas/esbuild-app
// yarn
yarn create @ewas/esbuild-app
// pnpm 
pnpm create @ewas/esbuild-app
```
