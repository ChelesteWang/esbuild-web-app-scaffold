A complete web development launcher based on Esbuild. Create web apps with no extra configuration.

## Quick Overview

```js
// npm
npm init @ewas/esbuild-app
// yarn
yarn create @ewas/esbuild-app
// pnpm 
pnpm create @ewas/esbuild-app
```
